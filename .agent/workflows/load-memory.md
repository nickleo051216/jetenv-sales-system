# 開啟專案記憶

當使用者提到 JetEnv 業務系統或相關功能時，請先執行以下步驟：

1. 讀取專案記憶文件
   ```
   view_file: C:\Users\Nick\.gemini\antigravity\scratch\jetenv-sales-system\.agent\memory.md
   ```

2. 根據記憶文件中的資訊來理解：
   - 專案技術棧和架構
   - 已確認的決策（如 LINE Bot 設定）
   - 資料庫結構
   - 目前的開發進度和待辦事項

3. 基於這些背景資訊回答用戶問題或執行任務
